# Risk Hotspots

Date: 2026-06-19

## High

1. 平台依赖风险
   - Seedance / Dreamina 能力、参数、CLI 形态可能变化。
   - 抖音创作者平台 DOM 和登录流程会漂移。

2. 许可证风险
   - `douyin-upload-mcp-skill` 使用 `AGPL-3.0`。
   - 在未隔离边界前直接吸收代码，会污染发布策略。

3. 交互风险
   - 抖音登录存在扫码、短信验证等多轮交互，不适合被伪装成单步无状态调用。

4. 合规风险
   - 生成内容可能触发平台内容审核、版权审核、真人脸限制。

## Medium

1. 通用性风险
   - 如果把 Skill 写成重度绑定某一 Agent 或某个 CLI，就无法满足“各种 AI 使用”。

2. 质量风险
   - 脚本、分镜、任务规格若无统一 schema，后续执行和发布层会碎裂。

3. 成本风险
   - 视频生成和重复重试存在积分/算力消耗，需要先有 dry-run 和预算提示。
