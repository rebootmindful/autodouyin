## Planned Flow

目标工作流以“可中断、可审计、可替换适配器”为原则：

1. 用户输入 brief / 目标
2. brief 规范化
3. 剧本生成
4. 分镜生成
5. Seedance 任务规格生成
6. 可选：调用执行适配器生成视频
7. 可选：调用抖音发布适配器上传并发布
