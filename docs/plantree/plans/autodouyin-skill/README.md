# autodouyin-skill

## Scope

定义一个可发布、可移植、可被多种 AI 使用的 Skill：

- 输入用户目标
- 生成视频脚本与分镜
- 生成 Seedance 2.0 可执行任务
- 在配置适配器后自动生成视频
- 自动上传并发布到抖音
