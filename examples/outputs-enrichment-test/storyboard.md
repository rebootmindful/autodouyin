# Storyboard

## 全局锁定
- Bauhaus minimalism meets zen emptiness, matte surfaces, geometric clarity, primary color accents, generous whitespace, functional beauty
- 同一角色保持一致五官、身份、服装和发型，不变脸、不换人。
- 同一场景保持一致环境、光线、背景与空间关系，不突然切换。

## 负面约束
- 禁止风格漂移
- 禁止角色变脸或换人
- 禁止突然偏色
- 禁止新增无关人物
- 禁止光线突变
- 禁止出现文字、字幕、LOGO、水印
- 禁止过度装饰
- 禁止渐变滥用
- 禁止无意义动效

## shot-01 0-3s
- Scene：scene-01
- 阶段：single-sequence
- 目的：hook
- 画面：竖屏中央，五张分镜卡片以 Bauhaus 网格瞬间排列成型，哑光表面反射柔和均匀光线，大面积留白衬托几何秩序，原色红点标记关键节点。
- 镜头：快速推近，从中景到近景锁定中央卡片组，镜头编码 Z4 Y4 X1，35mm，shallow 景深，第一眼抓住几何排列的功能美。
- 编码：Z4 Y4 X1 / 35mm / shallow
- 动作：分镜卡片从无到有以克制动画出现，无多余动效，仅几何位移。
- 音频：短促 matte 敲击音，干净利落，立刻建立节奏。
- 素材：C01, S01, P01
- 衔接：结尾卡片组居中定格，下一镜头从同一卡片组展开。
