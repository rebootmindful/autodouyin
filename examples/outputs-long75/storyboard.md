# Storyboard

## 全局锁定
- clean tech cinematic，统一视觉风格，画面干净，避免风格漂移
- 同一角色保持一致五官、身份、服装和发型，不变脸、不换人。
- 同一场景保持一致环境、光线、背景与空间关系，不突然切换。

## 负面约束
- 禁止风格漂移
- 禁止角色变脸或换人
- 禁止突然偏色
- 禁止新增无关人物
- 禁止光线突变
- 禁止出现文字、字幕、LOGO、水印

- Scene：scene-01
- 阶段：scene-base
- 目的：world-build
- 画面：建立世界与空间关系，先让观众理解“做一个 75 秒的长流程视频，完整展示从 brief、脚本、分镜、视频任务到发布准备的全链路”发生在什么工作流环境中，界面与主体位置稳定。
- 镜头：稳定机位为主，先远后中，镜头编码 Z7 Y4 X2 到 Z5 Y4 X2，50mm，deep景深。
- 编码：Z7 Y4 X2 / 50mm / deep
- 动作：主体进入场景，工作流逐步展开，但不急于爆发变化。
- 音频：节奏较慢的环境音和轻量配乐，优先可读性。 这是第 1/3 段，应保持风格、角色外观和场景关系不漂移。
- 素材：C01, S01, P01
- 衔接：当前 scene 内部可续拍；scene 与 scene 之间改用后期拼接而不是跨 scene 延长。
