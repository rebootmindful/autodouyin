# Storyboard

## 全局锁定
- clean tech premium，统一视觉风格，画面干净，避免风格漂移
- 同一角色保持一致五官、身份、服装和发型，不变脸、不换人。
- 同一场景保持一致环境、光线、背景与空间关系，不突然切换。

## 负面约束
- 禁止风格漂移
- 禁止角色变脸或换人
- 禁止突然偏色
- 禁止新增无关人物
- 禁止光线突变
- 禁止出现文字、字幕、LOGO、水印

## shot-01 0-1s
- Scene：scene-01
- 阶段：single-sequence
- 目的：hook
- 画面：开场直接打产品或应用结果镜头，画面突出finished result、clean UI，先给强展示感。
- 镜头：快速推近，第一眼抓人，镜头编码 Z4 Y4 X1，35mm，shallow景深。
- 编码：Z4 Y4 X1 / 35mm / shallow
- 动作：产品界面或结果卡片直接打在第一镜头，避免铺垫过长。
- 音频：短促电子提示音，开场即抓注意力。
- 素材：C01, S01, P01
- 衔接：保持主角、工作台和界面布局连续，后续段落沿用统一风格与空间关系。
