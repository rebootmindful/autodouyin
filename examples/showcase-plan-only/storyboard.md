# Storyboard

## 全局锁定
- Bauhaus minimalism meets zen emptiness, matte surfaces, geometric clarity, primary color accents, generous whitespace, functional beauty，统一视觉风格，画面干净，避免风格漂移
- 同一角色保持一致五官、身份、服装和发型，不变脸、不换人。
- 同一场景保持一致环境、光线、背景与空间关系，不突然切换。
- 主角保持同一张脸、同一发型、同一服装、同一年龄感和同一职业气质，不切换人物设定。

## 负面约束
- 禁止风格漂移
- 禁止角色变脸或换人
- 禁止突然偏色
- 禁止新增无关人物
- 禁止光线突变
- 禁止出现文字、字幕、LOGO、水印
- 禁止过度装饰
- 禁止渐变滥用
- 禁止无意义动效
