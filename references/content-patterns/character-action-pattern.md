# Character Action Content Pattern

character-action 的视觉哲学：**姿态即叙事，动势即节奏**。

## Type DNA

- 节奏：动作驱动节奏，快-慢-快-定格，不是匀速推进
- 视角：围绕角色的动态视角，观众 = 见证者
- 核心张力：姿态建立 -> 能量聚合 -> 动作爆发 -> 高潮定格
- 禁区：不要信息密度，不要 UI 展示，不要缓慢叙事

## Purpose 序列

### 短视频 (<= 15s)

| 顺序 | Purpose | 时长占比 | 动作任务 |
|------|---------|----------|----------|
| 1 | hook | 2-3s | 角色亮相：给姿态、给目标、立住人物 |
| 2 | setup | 3-4s | 能量聚合：界面元素/力量受控聚集，蓄力感 |
| 3 | turn | 3-4s | 动作推进：动作与镜头一起升级，速度感爆发 |
| 4 | payoff | 3-4s | 高潮击中：高速动作与信息流汇聚，最强完成镜头 |
| 5 | close | 2-3s | 英雄定格：背面或正面收束姿态，力量感残留 |

### 长视频 (16-60s)

动作型长视频用"蓄力-爆发-再蓄力-终极爆发"的波浪结构：
- world-build: 角色出场 + 环境建立 + 第一轮蓄力
- change-push: 第一波动作释放 + 新挑战出现 + 再次蓄力
- peak-moment: 终极动作爆发，最强视觉冲击
- emotional-close: 英雄定格，尘埃落定

## 视觉词汇

### 画面特征

- 角色是画面绝对中心，占画面 50-70%
- 姿态表达 > 细节表达：剪影感、轮廓光、动态模糊
- 能量可视化：粒子流、光线汇聚、色彩脉冲、界面元素飞舞
- 动作轨迹可视化：运动模糊、残影、速度线
- 每个镜头有一个明确的"力量方向"（上/下/左/右/向中心/向外扩散）

### 光线与能量

| 阶段 | 光线特征 | 效果 |
|------|----------|------|
| hook | 轮廓光/逆光，角色剪影或半剪影 | 神秘感 + 力量暗示 |
| setup | 光源从四周向角色汇聚 | 蓄力感，能量聚集 |
| turn | 角色自身开始发光/发亮，环境光跟随 | 力量释放，主导权转移 |
| payoff | 全画面高亮，粒子/光线爆发 | 视觉冲击最高点 |
| close | 光线回收到角色轮廓，背景暗下来 | 英雄定格，力量内敛 |

### 色彩策略

- 主色从美学预设取
- 能量/动作色：高饱和、与主色形成对比的强调色
- 蓄力段色彩逐渐加浓，爆发段色彩最饱和，收束段色彩回收
- 背景保持暗调或中性，不抢角色视觉焦点

## 音频设计

| Purpose | 声音特征 | 具体参考 |
|---------|----------|----------|
| hook | 低沉蓄力音，角色出场仪式感 | 低频 sub-bass + 金属共鸣 + 风声 |
| setup | 能量聚集声，频率逐渐升高 | 粒子汇聚音 + 电磁充能 + 节奏鼓点渐入 |
| turn | 动作爆发，声音与画面同步加速 | 打击音 + whoosh + 快速鼓点 + 配乐上扬 |
| payoff | 击中感最强音，全频段饱满 | bass drop + 确认击中音 + 配乐最高点 |
| close | 力量残留，低频余震 | 低频 tail + 回声渐远 + 最后一下心跳 |

关键技巧：**payoff 前留 0.2-0.3s 的"吸气"** -- 声音突然抽空再爆发，强化击中感。

## 相机倾向

character-action 的相机最"动"，但必须可控：

- hook: 中景 (Z4-Z5)，仰拍 (Y2-Y3) 给角色力量感，正面或侧 45
- setup: 缓慢环绕或推近 (Z5->Z4)，能量向角色汇聚的视觉引导
- turn: 加速运动 -- 快速推近 + 轻微角度变化，与动作方向一致
- payoff: 近景 (Z3)，85mm，动作充满画面，shallow 景深隔离背景
- close: 缓慢拉远 (Z5-Z6)，微仰 (Y3)，英雄定格

与 product-demo 的区别：相机运动幅度更大，允许仰拍和角度变化。
与 narrative-story 的区别：相机跟着动作走，不跟着情绪走。

## Prompt 范例

### hook (0-3s)

```
角色动作展示，强调姿态建立、动作升级和高潮定格。
风格锁定：[预设 seedance_style_anchor]。
0-3秒画面：[角色] 以 [姿态描述] 居中亮相，[预设光线但偏逆光/轮廓光]
勾勒角色轮廓，背景 [预设色板暗调]。角色周围有微弱的 [能量/粒子] 流动。
镜头：中景，微仰角度，35mm，medium 景深，角色力量感可感知。
动作：角色从静止进入 [准备姿态]，动作克制但充满势能。
声音：低频蓄力音，金属共鸣，建立仪式感。
额外要求：动作动势连续，角色姿态强，高潮击中感明确。
```

### payoff (9-12s)

```
角色动作展示。风格锁定同上。
9-12秒画面：[角色] 进入最强动作段，[具体动作描述]，
[能量/粒子/光线] 从角色中心向外爆发，全画面高亮，
[预设强调色] 脉冲贯穿画面。动作轨迹以 [运动模糊/残影] 可视化。
镜头：近景推近，85mm，shallow 景深，动作充满画面。
动作：[高潮动作 -- 击中/释放/完成] 的瞬间定格 0.5s。
声音：bass drop 击中音，全频段饱满，配乐最高点。
额外要求：动作动势连续，角色姿态强，高潮击中感明确。
```

## 常见错误

- hook 不要让角色静止站立 -- 即使静止也要有"势能"
- setup 不要变成信息展示 -- 蓄力是动作的前奏，不是解说
- turn 不要匀速 -- 动作要有加速感，从慢到快
- payoff 不要模糊 -- 高潮瞬间必须有 0.3-0.5s 的"定格感"
- close 不要松懈 -- 英雄定格仍要有力，不是"累了休息"
- 全片相机不要静止 -- 即使画面主体不动，相机也要有微动维持张力
