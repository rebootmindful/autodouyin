# Narrative Story Content Pattern

narrative-story 的视觉哲学：**情绪起伏可感知，人物状态变化可看见**。

## Type DNA

- 节奏：剧情短片节奏，不赶，给情绪留呼吸空间
- 视角：主角视角或紧贴主角的第三人称，观众 = 共情者
- 核心张力：压力建立 -> 转折介入 -> 状态逆转
- 禁区：不要产品卖点感，不要信息密度堆叠，不要纯展示

## Purpose 序列

### 短视频 (<= 15s)

| 顺序 | Purpose | 时长占比 | 情绪任务 |
|------|---------|----------|----------|
| 1 | hook | 2-3s | 压力先立住：主角困境、截止压力、未完成状态 |
| 2 | setup | 3-4s | 问题具象化：桌面、时间、人物状态，让观众感受到"来不及了" |
| 3 | turn | 3-4s | 介入与逆转：AI/工具/力量登场，局面开始翻转 |
| 4 | payoff | 3-4s | 主角状态变化：从焦虑到放松，从混乱到有序 |
| 5 | close | 2-3s | 情绪落点：停在成功前一刻或完成后的呼吸瞬间 |

### 长视频 (16-60s)

三阶段结构，情绪弧线拉长：
- world-build: 主角的世界、日常、困境的全貌展开
- change-push: 介入力量逐步改变局面，但仍有阻力
- emotional-close: 最终逆转完成，主角新状态定格

关键：长视频的 turn 不是一次性事件，而是渐进过程。

## 视觉词汇

### 画面特征

- 主角是画面绝对中心，环境服务于人物状态表达
- 情绪可视化：焦虑 = 杂乱/暗调/拥挤，释放 = 整洁/亮调/留白
- 时间压力可视化：倒计时、堆叠文件、红色标记
- 转折可视化：光线变化、色彩温度变化、空间打开
- 每个镜头的情绪状态必须明确可辨

### 光线与情绪映射

| 阶段 | 光线特征 | 色温 |
|------|----------|------|
| hook | 压迫感布光，侧光或顶光制造阴影 | 偏冷或偏暗 |
| setup | 环境光不均匀，暗示不安定 | 冷调主导 |
| turn | 光线开始变化，新光源出现 | 冷暖过渡 |
| payoff | 均匀明亮主光，阴影退散 | 暖调主导 |
| close | 柔和自然光，呼吸感 | 温暖舒适 |

### 空间表达

- 压力段：空间拥挤，元素靠近画面边缘，前景遮挡
- 释放段：空间打开，留白增加，画面通透
- 这个从"紧"到"松"的空间变化是 narrative-story 的核心视觉叙事手段

## 音频设计

| Purpose | 声音特征 | 具体参考 |
|---------|----------|----------|
| hook | 压迫感环境音或紧迫节奏 | 低频嗡鸣 + 时钟滴答 + 紧张弦乐 |
| setup | 焦虑感叠加，声音层次变密 | 键盘敲击加速 + 纸张翻动 + 低频渐强 |
| turn | 声音断裂或变化，新音色进入 | 突然静音 0.5s -> 清澈音色进入 (钢琴/合成器) |
| payoff | 情绪释放，配乐最高点 | 温暖弦乐或明亮合成器 pad + 满足感尾音 |
| close | 安静收束，留呼吸空间 | 配乐渐弱 + 环境音回归 (鸟鸣/风声/咖啡杯) |

关键技巧：**turn 之前留一个静音间隙** (0.3-0.5s)，让转折有听觉上的"呼吸"。

## 相机倾向

narrative-story 的相机服务于情绪，不是信息：

- hook: 中近景 (Z4-Z5)，稍高角度 (Y5) 俯视主角，制造压迫感
- setup: 中景 (Z5)，手持微晃或不稳定感，暗示焦虑
- turn: 角度变化 -- 从俯视切到平视或微仰，暗示力量转移
- payoff: 近景 (Z3-Z4)，平视主角面部或成果，85mm 浅景深隔离背景
- close: 缓慢拉远 (Z6)，平视或微仰，给主角留空间

与 product-demo 的区别：相机运动更情绪化，允许不完美和手持感。

## Prompt 范例

### hook (0-3s)

```
剧情短片风格，强调情绪起伏、人物状态变化和结果反转。
风格锁定：[预设 seedance_style_anchor]。
0-3秒画面：[主角] 处于 [压力场景]，[预设光线但偏暗/偏冷] 制造压迫感，
桌面堆满 [压力道具]，画面空间拥挤，前景有遮挡元素。
主角表情/姿态传达 [焦虑/紧迫/困境]。
镜头：中近景，稍高角度俯视，50mm，medium 景深，环境压力可读。
动作：主角 [具体压力动作 -- 揉太阳穴/快速翻文件/盯着屏幕发呆]。
声音：低频压迫感环境音，隐约时钟滴答声，建立紧迫节奏。
额外要求：人物情绪递进，压力到释放，剧情反转明确。
```

### turn (6-9s)

```
剧情短片风格。风格锁定同上。
6-9秒画面：[介入力量] 出现，光线开始变化 -- [新光源描述] 从 [方向] 打入，
色温从冷调开始向暖调过渡。桌面 [压力道具] 开始被 [清理/替代/转化]。
主角姿态从 [焦虑] 转为 [专注/惊讶/放松]。
镜头：角度从俯视切为平视，轻微推近，50mm，medium 景深。
动作：[介入动作 -- 界面出现/工具启动/结果生成]，元素开始有序排列。
声音：短暂静音后清澈音色进入 (钢琴单音或合成器 pad)，节奏开始上扬。
额外要求：人物情绪递进，压力到释放，剧情反转明确。
```

## 常见错误

- hook 不要给解决方案 -- 先让观众感受到压力
- setup 不要跳过 -- 问题不具象化，转折就没有力量
- turn 不要太快 -- 给变化留 3-4 秒，不要一秒翻转
- payoff 不要只展示结果 -- 必须展示主角的**状态变化**
- close 不要加新信息 -- 这是情绪着陆的时刻，不是总结
- 全片不要一直暗或一直亮 -- 光线变化就是叙事弧线
